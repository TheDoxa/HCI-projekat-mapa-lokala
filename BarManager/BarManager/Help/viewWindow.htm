View window

1. Add - open form for adding new bar, bar type or bar label depends on selected menu item from point 5.

2. Edit - open form for editing selected bar, bar type or bar label.

3. Remove - remove selected bar, bar type or bar label.

4. Filter - open new dialog for setting filter criteria. After applying criteria, map will refresh.

5. Menu - select options are bar, bar type and bar label. Tabel (point 7) will display those entites based on selected item from this menu.

6. Search - enter text in field (which refers to the description or name of bars) and after pressing enter button or click on loop icon, table will refresh.

7. Display bars/bar types/bar labels depending on selected item from menu (point 5).

Shortcuts

F1 - Help

F6 - Edit selected item

F7 - Delete selected item

Esc - Close window
