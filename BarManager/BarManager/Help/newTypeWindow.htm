Bar type

Bar type contains: Id, Name, Description and Icon.

Bar type window

1. ID - identifier for bar type. Must be unique.

2. Auto gen - by clicking on this button, program generates the lowest available ID for bar type.

3. Add - collect all fields, create new bar type and save it. Warning dialog will show if entered ID is already in use.

4. Choose image - select image from your computer.
