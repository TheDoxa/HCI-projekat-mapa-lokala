Bar

Bar contains: Id, Name, Description, Type, Alcohol status, Icon, Handicapped adapted, Smoking allowed, Reservation allowed, Price category, Max capacity, Open date, Labels.

Bar window

1. ID - identifier for bar. Must be unique.

2. Auto gen - by clicking on this button, program generates the lowest available ID for bar.

3. Name - must contain at least 3 letters.

4. Add new type - open window for adding new type, after type is added type combobox will add new type in collection.

5. Max capacity - capacity value must be between 1 and 2000.

6. Choose image - select image from your computer.

7. Default image - set image depends on type image.

8. add label - select label from available labels collection and after click on 'add label' button, selected label will move from available labels collection to chosen labels collection.

9. remove label - select label from chosen labels collection and after click on 'remove label' button, selected label will move from chosen labels collection to available labels collection.

10. Add bar - collect all fields, create new bar and save it. Warning dialog will show if entered ID is already in use.

11. Cancel - close the form for adding new bar.

Shortcuts
