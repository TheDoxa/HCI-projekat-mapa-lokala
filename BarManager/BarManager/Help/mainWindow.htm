Main window

1. Help - open help window.

2. Bar list - display all bars. Bars from this list can be moved and shown on map.

3. Filter - open new dialog for setting filter criteria. After applying criteria, map will refresh.

4. Search - enter text in field (which refers to the description or name of bars) and after pressing enter button or click on loop icon, map will refresh.

5. Main map - display selected map and all bars dropped on the map. Also, map can be scrolled.

6. Display of all maps - display list of all maps in small size. After clicking on one of them, selected one will be positioned on main map.

Shortcuts

F1 - Help

F2 - Add new bar

F3 - Add new type

F4 - Add new label

F5 - View all bars, types and labels
